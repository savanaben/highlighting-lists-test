Adjacent bullets highlighted

Each highlighted li draws a background/border using ::before. CSS adjacency selectors and :has(+ li.li-highlight) detect neighboring highlighted items, then remove shared borders (border-bottom: none on the upper item and border-top: none on the lower one) so the run appears as one continuous highlight.

Progressing highlighting of a list

This section is to just define the content rule of how to highlight a list progressively. Each li should get the highlight in this situation (when there's only 1 bullet, the first one, we are not using the spaced ul/ol whole-list highlight, because we want the highlight style to be consistent across screens/new bullet introductions. This is something authors need to manage (what ICCI outline element gets the highlight applied)).
